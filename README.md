# QT를 사용한 은행 프로그램

## 기능

- **파일 입출력으로 데이터 불러오고 수정 가능**
- **회원가입 및 로그인**
- **로그인 시 관리자 회원을 구분하여 자동으로 넘어가는 페이지**  
  - 관리자 ID: `admin`, PW: `1234`
- **회원의 입금, 출금, 송금, 계좌 생성**
- **관리자 회원으로 로그인 시, 회원들의 입금, 출금, 송금을 로그로 기록하여 확인할 수 있음**
